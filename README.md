# Bilibili Automatic Dark Mode Switching

> **让 B 站网页版原生深色模式真正实现“自动化”。不再手动切换，拒绝进站闪白。**

---

## ✨ 核心优势 (Advantages)

* **🛡️ 100% 原生支持**：不使用暴力 CSS 覆盖，而是通过脚本触发 B 站官方的深色主题逻辑。这意味着所有 UI 细节（评论区、弹幕、动态）均由 B 站官方样式保证，完美兼容。
* **🚫 彻底告别闪白**：在页面首帧渲染前即注入主题配置。即使在弱网环境下，也能保证进站即黑，保护视力。
* **🖱️ 丝滑交互体验**：
    * **热更新**：切换模式无需刷新页面，实时生效。
    * **可拖拽面板**：右下角悬浮窗支持自由拖动，并自动保存位置。
    * **一键隐藏**：点击面板右上角 `×` 即可在本次会话中隐藏，保持页面清爽。
* **🤖 智能冲突检测**：如果你手动在 B 站原生菜单切换了主题，脚本会识别并暂时“闭嘴”，优先尊重你的手动选择。
* **⏰ 灵活切换模式**：
    * **跟随系统**：与 Windows/macOS 系统深色设置实时同步。
    * **定时模式**：自定义起止时间（如 22:00 至次日 06:00）。
    * **手动关闭**：一键停用脚本干预。

---

## ⚠️ 已知限制 (Limitations)

* **无法切回浅色**：通过自动切换的深色模式无法正常切回浅色（除非刷新）
* **SPA 跳转延迟**：B 站部分单页应用（SPA）组件在极少数情况下可能由于内部状态未及时响应，导致局部颜色稍微慢半拍，刷新即可解决。
* **环境依赖**：脚本深度依赖 `GM_setValue` 等高级 API，建议在 **Tampermonkey** 或 **ScriptCat** 完整版环境下运行。
* **隐藏机制**：点击 `×` 后，悬浮窗在当前标签页生命周期内不再出现，需刷新页面或打开新链接才会重新加载。

## 🚀 安装说明 (Installation)

你可以通过以下两种方式安装脚本：

### 方法 A：直接拖拽（推荐）
1.  下载本项目中的 `.js` 后缀脚本文件。
2.  打开浏览器的插件管理页（油猴/脚本猫的主界面）。
3.  将下载的文件**直接拖入**插件管理界面，点击安装即可。

### 方法 B：手动复制代码
1.  打开本项目中的脚本文件代码。
2.  在油猴/脚本猫中点击“新建脚本”。
3.  删除默认代码，将本项目代码**全部粘贴**进去并保存 (Ctrl + S)。

## 🛠️ 技术原理
脚本通过拦截并模拟 `StorageEvent` 来与 B 站的 React/Vue 框架通信，同时在 `document-start` 阶段锁定 `Cookie` 和 `LocalStorage` 中的 `theme_style` 字段，从而实现比官方更智能的自动化切换。
